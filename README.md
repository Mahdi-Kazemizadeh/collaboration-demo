# Collaboration Demo Project

This project is used to practice Git and GitHub collaboration workflows.

## How to contribute
- Open an issue
- Create a branch
- Submit a pull request

## Collaboration workflow
1. Create or choose an issue.
2. Create a new branch for the issue.
3. Make changes and commit them.
4. Open a pull request.
5. Wait for review before merging.

Each pull request should be reviewed before it is merged into the main branch.
